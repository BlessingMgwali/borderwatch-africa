# BorderWatch Africa 🚛

**The leading logistics intelligence platform 
for Southern African transport operators.**

## Live Demo
[View Live Site](https://bolt.new/~/sb1-qhcxgvfb)

## What It Does
BorderWatch Africa helps transport companies:
- Monitor real-time border congestion across 18 SADC border posts
- Track fleet movements across Southern Africa
- Find return load opportunities (freight marketplace)
- Reduce fuel costs through AI route optimization
- Manage cross-border logistics operations

## Tech Stack
- React 18 + TypeScript
- Tailwind CSS + Shadcn UI
- Framer Motion (animations)
- Recharts (data visualization)
- React Router (navigation)
- Claude AI API (business intelligence)
- PayFast (payment processing)

## Key Features
- Multi-tenant SaaS architecture
- Role-based access control (Super Admin, Company Admin, Driver, Cargo Owner)
- Two-sided freight marketplace
- Automated 10% commission system
- AI-powered business analytics
- Real-time border status monitoring
- PayFast subscription billing in ZAR
- Responsive design (mobile + desktop)
- Multi-industry support (Mining, Agriculture, Construction, Retail, Cold Chain)

## User Roles
| Role | Access |
|------|--------|
| Super Admin | Full platform control + revenue dashboard |
| Company Admin | Their company data only |
| Driver | Border status + condition reporting |
| Cargo Owner | Post shipments + receive quotes |

## Business Model
| Stream | Details |
|--------|---------|
